GenerateFileList.ps1

打开powershell ，拖进去

```
# 获取当前脚本所在目录（或可改为指定目录）
$targetDir = Split-Path -Parent $MyInvocation.MyCommand.Definition
# 如果你想固定某个目录，把上面一行注释掉，用下面这行并修改路径：
# $targetDir = "D:\25515\OneDrive\dushu\星球\星辰天机阁SVIP专属合集\孙子兵法52篇（已完结）"

$outputFile = Join-Path $targetDir "file_list.txt"

Get-ChildItem -Path $targetDir -Recurse -File | ForEach-Object {
    "$($_.FullName)  ($($_.Length) bytes)"
} | Out-File -FilePath $outputFile -Encoding UTF8

Write-Host "文件列表已生成：$outputFile" -ForegroundColor Green
Pause
``
